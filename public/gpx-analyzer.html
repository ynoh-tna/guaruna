Private by design

The GPX file is read and analysed entirely in your browser with JavaScript — it is never sent to any server. The map tiles come from OpenStreetMap; only the map view you see is requested, never your file.

Where to get a GPX file

Export one from Garmin Connect, Strava (open an activity → "Export GPX"), Komoot, Wahoo or Coros. Or grab a ready-made route from the GPX database.

Frequently asked questions

Is my GPX file uploaded anywhere?

No. The file is read and analysed entirely in your browser with JavaScript, so it is never sent to a server.

Is the GPX viewer free?

Yes, completely free, with no account and no file limits.

What can I see from a GPX file?

Your route on an OpenStreetMap map, total distance, elevation gain and loss, moving pace when the file has timestamps, an elevation profile, and per-kilometre splits.

Where can I get a GPX file?

Export one from Garmin Connect, Strava, Komoot, Wahoo or Coros, or grab a ready-made route from the Guaruna GPX database.
